# Retail Sales Analytics Dashboard

## Project Overview

This project analyzes retail sales transactions using Power BI.

The dashboard provides insights into:

- Revenue Performance
- Profitability
- Product Analysis
- Customer Segmentation
- Payment Method Trends
- Monthly Revenue Trends

## Tools Used

- Power BI
- SQL Concepts
- DAX

## Key KPIs

- Total Revenue
- Total Profit
- Total Orders
- Total Customers

## Business Insights
